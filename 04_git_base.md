# Git基本使用

## 1. 获取仓库

#### 1.1 创建仓库

```shell
$ git init git_tutorial
或者
$ mkdir repository    # 创建仓库目录
$ cd repository       # 进入到仓库目录
$ git init			 # 创建仓库
```

#### 1.2 检出仓库

```shell
$ git clone git@host:/path/to/repository 
$ git clone git@host:/path/to/repository rename_repository # clone 之后重命名仓库
```
